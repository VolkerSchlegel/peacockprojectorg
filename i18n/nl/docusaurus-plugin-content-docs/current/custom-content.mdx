## Elusive Targets

<details>

<summary>Click to Expand</summary>

Within Peacock, you can replay Elusive Targets without any time limit on restarts (unlike Elusive Target Arcade), and without a time limit.
  * **How to Play**: After starting up Hitman 3 in Peacock, go to `Career` -> `Elusive Targets` in-game.

</details>

## Escalations

<details>
<summary>Click to Expand</summary>

* **Custom Peacock Escalations**
  * Peacock supports custom (made with user input) escalations. Deze escalations zijn standaard beschikbaar in het spel. Je kan ook enkele aangepaste escalations zien in de [Roadmaps](https://thepeacockproject.org/wiki/roadmaps).

* **Officiële Server Escalations**
  * Peacock ondersteunt ook enkele officiële server escalations, hoewel ze niet allemaal geport zijn. Deze escalations zijn standaard beschikbaar in het spel.
* **Legacy Escalations**
  * Peacock ondersteunt Hitman 2016 escalations die niet zijn overgezet naar Hitman 2 of 3 (legacy escalations).
  * To install legacy escalations, search `legacy in: plugin-releases` in the Peacock Discord. Download `LegacyEscalationsPlugin.js`, install it to your main Peacock install folder (`...Peacock-vx.x.x/Peacock-vx.x.x`).
* **How to Play Escalations**: Go to `Destinations` menu, then to a specific map in-game to access escalations.

</details>

## Contracten Modus

<details>
<summary>Click to Expand</summary>

Peacock supports both official server and custom Peacock contracts.

* **Featured contracten**
  * Dit omvat alle officiële contracten van de servers, evenals sommige aangepaste contracten die door het Peacock ontwikkelingsteam worden gecreëerd. Je kan enkele van deze op maat gemaakte contracten [here] zien(https://thepeacockproject.org/wiki/roadmaps).
  * Hoe te Spelen: Ga naar het ```Game Modes`` -> ```Contracts`` -> ```Featured``` menu in-game
* Door Gebruiker Gemaakte, Officiële Server Contracten**
  * Hoe te Installeren: Ga naar het ```Game Modes`` -> ```Contracts`` -> ```Contract Lookup`` menu in-game. Voer de officiële server contract ID in (let op: alleen Steam en Epic Hitman 3 contracten worden ondersteund). Het contract wordt nu gedownload in ```...Peacock-vx.x.x/Peacock-vx.x.x/Contracts/```.
  * Hoe te Spelen: Ga naar ```Game Modes`` -> ```Contracts`` -> ```My Contracts``` in-game om te spelen.
* **User Created, Peacock Contracts**
  * How to Install: Download the contracts ```.ocre``` file from the ```share-contracts``` channel in Peacock Discord. Installeer dit ```.ocre`` bestand in uw ```...Peacock-vx.x.x/Peacock-vx.x.x/Contracts`` map.
  * Hoe je je eigen maakt: Als je een contract maakt in-game terwijl je in Peacock server bent, zal je contract worden opgeslagen als een ```.ocre`` bestand in de ```...Peacock-vx.x.x/Peacock-vx.x.x/Contracts`` map. Zorg ervoor dat je het ```.ocre`` bestand deelt in het ``share-contracts`` kanaal van de Peacock Discord als je wilt dat anderen je aangepaste Peacock contract kunnen spelen.
  * Hoe te Spelen: Ga naar ```Game Modes`` -> ```Contracts`` -> ```My Contracts``` in-game om te spelen.

</details>

## Plugins

<details>
<summary>Click to Expand</summary>

Besides Legacy Escalations Plugin, which was described earlier in the Escalations section, Peacock supports a broad variety of  server side mods (plugins). Deze omvatten, maar zijn niet beperkt tot:
* Kill Everyone Mode
  * Alle echte NPC's gemarkeerd als doelwit, kan Silent Assassin rating terugkrijgen na het doden van iedereen.
  * Opmerking: Erg buggy voor maps met 200+ doelen, geweren kunnen willekeurig vastlopen. Misschien moet je beginnen met mensen melee te doden.
* Sarajevo Six
  * Bonusmissies die anders alleen beschikbaar zijn in de Hitman 2016 Playstation-versie.
  * Installatie-instructies: https://rentry.org/H3_TS6
* Plugins om anders ongebruikte wapens aan het spel toe te voegen
* Plugins om bestaande wapens en vermommingen aan te passen.
* **Hoe Plugins te Installeren**: In het algemeen, om plugins te installeren, ga naar ```#plugin-releases`` in de Peacock Discord en download een plugin waar je in geïnteresseerd bent. Installeer het dan in de hoofdmap van je Peacock installatie (```...Peacock-vx.x.x/Peacock-vx.x.x``). Zorg ervoor dat je kijkt voor verdere instructies in het ```#plugin-releases``` kanaal voor een bepaalde plugin, aangezien bepaalde plugins verdere installatiestappen vereisen of incompatibel zijn met nieuwere versies van Peacock.

**Opmerking: Plugins zijn slechts een klein deel van de Hitman modding scene. Veel Hitman mods worden client-side gedaan en vallen buiten het bereik van Peacock.** Client-side mods worden hier samengevoegd: https://www.nexusmods.com/hitman3

</details>
