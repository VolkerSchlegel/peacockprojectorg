O Peacock tem vários benefícios incluídos para nossos usuários. Aqui vai uma espiada no nosso conteúdo customizado!

<div class="make_h2_smaller">

## Alvos Elusivos

<details>

<summary>Click to Expand</summary>

Within Peacock, you can replay Elusive Targets without any time limit on restarts (unlike Elusive Target Arcade), and without a time limit.
  * **How to Play**: After starting up Hitman 3 in Peacock, go to `Career` -> `Elusive Targets` in-game.

</details>

## Escalações

<details>
<summary>Click to Expand</summary>

* **Custom Peacock Escalations**
  * Peacock supports custom (made with user input) escalations. Essas escalações estão disponíveis em jogo por predefinição. Você pode ver algumas das escalações customizadas listadas nos [Roadmaps](https://thepeacockproject.org/wiki/roadmaps) também.

* **Escalações dos Servidores Oficiais**
  * Peacock também suporta algumas escalações dos servidores oficiais, embora não todas foram portadas. Essas escalações estão disponíveis em jogo por predefinição.
* **Escalações Legacy**
  * Peacock suporta escalações do Hitman 2016 que não foram portadas para o Hitman 2 ou 3 (escalações 'legado').
  * To install legacy escalations, search `legacy in: plugin-releases` in the Peacock Discord. Download `LegacyEscalationsPlugin.js`, install it to your main Peacock install folder (`...Peacock-vx.x.x/Peacock-vx.x.x`).
* **How to Play Escalations**: Go to `Destinations` menu, then to a specific map in-game to access escalations.

</details>

## Modo Contratos

<details>
<summary>Click to Expand</summary>

Peacock supports both official server and custom Peacock contracts.

* **Contratos em Destaque**
  * Aqui estão incluídos todos os contratos em destaque dos servidores oficiais, assim como alguns contratos customizados criados pela equipe de desenvolvimento do Peacock. Você pode ver alguns desses contratos customizados [aqui](https://thepeacockproject.org/wiki/roadmaps).
  * Como Jogar: Vá para o menu ```Game Modes``` -> ```Contracts``` -> ```Featured``` em jogo
* **Criados Pelo Usuário, Contratos dos Servidores Oficiais**
  * Como baixar: Vá para o menu ```Game Modes``` -> ```Contracts``` -> ```Contract Lookup``` em jogo. Entre o ID do contrato do servidor oficial (nota: apenas contratos da Steam ou Epic são suportados). O contrato será depois disso baixado em ```...Peacock-vx.x.x/Peacock-vx.x.x/Contracts/```.
  * Como Jogar: Vá para ```Game Modes``` -> ```Contracts``` -> ```My Contracts``` para jogar.
* **User Created, Peacock Contracts**
  * How to Install: Download the contracts ```.ocre``` file from the ```share-contracts``` channel in Peacock Discord. Coloque esse arquivo ```.ocre``` na sua pasta ```...Peacock-vx.x.x/Peacock-vx.x.x/Contracts```.
  * Como Criar o Seu Próprio: Se você criou um contrato em jogo no servidor Peacock, seu contrato será salvo como um arquivo ```.ocre``` na pasta ```...Peacock-vx.x.x/Peacock-vx.x.x/Contracts```. Tenha a certeza de compartilhar seu arquivo ```.ocre``` no canal ```share-contracts``` do servidor Discord do Peacock se quiser que outros joguem seu contrato customizado Peacock.
  * Como Jogar: Vá para ```Game Modes``` -> ```Contracts``` -> ```My Contracts``` para jogar.

</details>

## Plugins

<details>
<summary>Click to Expand</summary>

Besides Legacy Escalations Plugin, which was described earlier in the Escalations section, Peacock supports a broad variety of  server side mods (plugins). Esses incluem, mas não estão limitados a:
* Modo Mate Todo Mundo
  * Todos os NPCs reais são marcados como alvos, e você pode ter avaliação Assassino Silencioso depois de ter matado a todos.
  * Nota: Bem bugado em mapas com 200+ alvos, armas podem falhar aleatoriamente. Você pode precisar matar usando métodos corpo a corpo.
* Os Seis de Sarajevo
  * Missões Bônus de outra forma disponíveis apenas na versão PlayStation de Hitman 2016.
  * Instruções de Instalação: https://rentry.org/H3_TS6
* Plugins para adicionar armas não utilizadas ao jogo
* Plugins para modificar armas existentes e disfarces.
* **Como Instalar Plugins**: Geralmente, para baixar plugins, vá para ```#plugin-releases``` no servidor Discord do Peacock e faça o download de um plugin em que tenha interesse. Então, mova para a pasta principal de instalação do Peacock (```...Peacock-vx.x.x/Peacock-vx.x.x```). Tenha certeza de verificar por quaisquer outras instruções no canal ```#plugin-releases``` por um certo plugin, já que certos plugins poder necessitar de mais passos de instalação ou são incompatíveis com versões mais recentes do Peacock.

**Nota: Plugins são apenas uma pequena porção da comunidade Hitman de mods. Muitos mods Hitman são de responsabilidade do usuário e fora do alcance do Peacock.** Mods de responsabilidade do usuário são agregados aqui: https://www.nexusmods.com/hitman3

</details>
